# Kakaotalk Stella Sora Theme
## Introduction
Stella Sora Theme provides Stella Sora style theme to Kakaotalk application.
## Download & Install
1. Go to Releases and download latest APK file
2. Install and run APK file
3. If you already installed Kakaotalk, click Apply.
4. If you don't have Kakaotalk, click Install Kakaotalk. After Kakaotalk is installed, re-run the app and click Apply.

# 카카오톡 스텔라 소라 테마
## 개요
스텔라 소라 테마는 카카오톡에 스텔라 소라 스타일 테마를 적용합니다.
## 다운로드 및 설치
1. Latest Release에서 APK 파일을 다운로드합니다.
2. 다운로드 받은 APK 파일을 설치하고 엽니다.
3. 카카오톡이 이미 설치되어 있다면, 적용을 누릅니다.
4. 카카오톡이 설치되어 있지 않다면, 카카오톡 설치를 눌러 카카오톡을 설치한 후, 다시 앱으로 돌아와 적용을 누릅니다.
## 저작권
이 테마의 모든 리소스 이미지들의 저작권은 카카오 및 요스타가 소유합니다.
